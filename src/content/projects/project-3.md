## Overview

An interactive web application built with R Shiny that lets users explore the mechanics of linear regression by manipulating data and model parameters in real time.

## Features

- Adjust sample size, noise level, and slope interactively
- Visualise the fitted regression line and confidence intervals
- Inspect residuals and model diagnostics
- See how assumptions like homoscedasticity are affected by data changes

## Purpose

This app was built as an educational tool — making abstract statistical concepts tangible through direct manipulation. It's particularly useful for students learning regression for the first time.
